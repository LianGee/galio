# Galio

`galio` 是一款针对在`k8s`上发布应用的工具。`galio`支持模板式发布，您可以设定好模板后，`galio`会通过模板构建相应的发布信息，通过`api`进行发布。

## 功能  

1. 构建
2. 发布
3. 数据库查询
